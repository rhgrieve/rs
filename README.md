# rs

An `ls` clone in rust

```bash
USAGE:
  rs [OPTIONS] PATH

OPTIONS:
  -h, --help
  -v, --version
  -a, --all
  -A, --almost-all
  -1, --one-line
  -l, --long
  -n, --numeric-uid-gid
  -H, --human-readable
  --group-directories-first
  -B, --ignore-backups
  -t, --sort-time
  -s, --size
  -S, --sort-size
  -i, --inode
  -k, --kibibytes
```
